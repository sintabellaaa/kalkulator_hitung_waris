# kalkulator_hitung_waris

Aplikasi berbasis **Streamlit** untuk menghitung pembagian harta warisan sesuai aturan **Faraidh** dalam Islam (QS. An-Nisa ayat 11-12).  
Dibuat sebagai bagian dari tugas mata kuliah *sains dan teknologi dalam islam*.

---

## 🚀 Fitur
- Input total harta warisan
- Pilihan ahli waris (suami, istri, ayah, ibu, anak)
- Perhitungan otomatis sesuai aturan Faraidh
- Tampilan hasil pembagian lengkap dengan persentase
- Mode tampilan sederhana dan interaktif
